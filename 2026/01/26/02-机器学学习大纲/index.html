    5. 1.5. 资源
      1. 1.5.1. 免费微课程：Kaggle Learn “Intro to Machine Learning”
      2. 1.5.2. 图书：《Hands-On Machine Learning》第 2 章～第 6 章
  2. 2. 阶段 2：算法深化 + 竞赛刷榜（6-18 周）
    1. 2.1. 目标：能独立做特征工程、调参、集成学习，冲进 Kaggle 前 20%。
    2. 2.2. 技能
      1. 2.2.1. 模型家族：GBDT（XGBoost/LightGBM）、SVM、聚类、PCA、Ensemble
      2. 2.2.2. 特征工程：目标编码、频次编码、嵌入、时序滑窗
      3. 2.2.3. 超参优化：Bayesian Optimization、Optuna
    3. 2.3. 工具
      1. 2.3.1. LightGBM 4.x、Optuna、CatBoost
    4. 2.4. 最小实战
      1. 2.4.1. “房价预测”或“店铺销量”比赛，单模型突破 Public LB 0.12 → 融合到 0.10
    5. 2.5. 资源
      1. 2.5.1. 图书：《机器学习实战》（李航）+《百面机器学习》
    6. 2.6. 经验帖：Kaggle Grandmaster 访谈
  3. 3. 阶段 3：深度学习与大模型（18-36 周）
    1. 3.1. 目标：理解反向传播/Transformer，能微调大模型，做出端到端 Demo。
    2. 3.2. 技能
      1. 3.2.1. CNN/RNN/Transformer 原理；注意力、位置编码、残差
      2. 3.2.2. 框架：PyTorch 2.x + Hugging Face Ecosystem
      3. 3.2.3. 任务：文本分类、NER、图文检索、RAG
    3. 3.3. 工具
      1. 3.3.1. LoRA/QLoRA、DeepSpeed、LangChain、Llama-Factory
    4. 3.4. 最小实战
      1. 3.4.1. 用 1 张 24 G 显卡微调 ChatGLM3-6B 做“公司 FAQ 问答”，接入企业微信
    5. 3.5. 资源
      1. 3.5.1. 图书：《深度学习入门：基于 Python 的理论与实现》
      2. 3.5.2. 视频：fast.ai 2025 版 Practical Deep Learning
  4. 4. 阶段 4：MLOps & 产业落地（36-52 周）
    1. 4.1. 目标：让模型长在生产环境，可灰度、可回滚、可监控。
    2. 4.2. 技能
      1. 4.2.1. CI/CD：GitHub Actions + Dockerfile + Kubernetes
      2. 4.2.2. 数据/模型版本管理：DVC、MLflow、Weights & Biases
      3. 4.2.3. 监控：Prometheus + Grafana，指标漂移检测
    3. 4.3. 工具
      1. 4.3.1. Kubeflow、MLflow 2.x、BentoML、Ollama
      2. 4.3.2. 最小实战
      3. 4.3.3. 把阶段 3 的 FAQ 模型封装成 REST/gRPC 服务 → 部署到阿里云 ACK → 配置滚动发布 & 监控
    4. 4.4. 资源
      1. 4.4.1. 免费课程：Full-Stack Deep Learning 2025
      2. 4.4.2. 手册：《MLOps 三步法》
    5. 4.5. 加餐模块（贯穿全年）
      1. 4.5.1. 数学补漏：线性代数（MIT 18.06）、概率（Stat110）、最优化（CS229-notes）
      2. 4.5.2. 论文复现：每月挑 1 篇“有代码”的顶会论文（ Papers with Code 榜单）
      3. 4.5.3. 开源贡献：向 Hugging Face、LangChain 提 PR，积累简历亮点
    6. 4.6. 学习节奏建议
      1. 4.6.1. 70 % 时间写代码跑实验；20 % 读文档/源码；10 % 看数学。
      2. 4.6.2. 每阶段输出一篇中文技术博客 + GitHub 仓库，形成作品集。
      3. 4.6.3. 每季度回顾路线图，根据招聘 JD 调整深度（如自动驾驶方向就重点冲多模态+BEV）。
      4. 4.6.4. 按上面 4 阶段稳扎稳打，一年即可从小白成长为“能独立交付模型+上线”的初级 ML Engineer；第二年再择方向深耕（CV、NLP、强化学习、AI 芯片等），自然水到渠成。祝你学习顺利！

02-机器学学习大纲

每阶段都标明：

  • ①要掌握的核心技能；
  • ②推荐工具/框架；
  • ③最小实战项目；
  • ④预计周期。你可以按需快进，也可循环迭代。

资源

阶段 2：算法深化 + 竞赛刷榜（6-18 周）
